# 📊 CCE Project – Flask Web Application

## 🌿 Overview

This project is a web-based application built using **Python Flask**.
It integrates a machine learning model to provide predictions based on user input.

The application demonstrates the combination of:

* Backend development using Flask
* Machine Learning model integration
* Web interface using HTML templates

---

## 🚀 Features

* User-friendly web interface
* Machine learning prediction system
* Fast and lightweight Flask backend
* Easy to run locally

---

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* HTML / CSS
* Pickle (for model loading)

## ⚙️ Installation & Setup

### Step 1: Clone the repository

```
git clone https://github.com/Someshkumar8998/CCE-Project-main.git
cd CCE-Project-main
```

### Step 2: Install required libraries

```
pip install flask scikit-learn numpy pandas
```

### Step 3: Run the application

```
python app.py
```

### Step 4: Open in browser

```
http://127.0.0.1:5000/
```

---

## 🧠 How It Works

* The user enters input through the web form
* Flask receives the request
* The trained model (`model.pkl`) processes the input
* Prediction result is displayed on the webpage
